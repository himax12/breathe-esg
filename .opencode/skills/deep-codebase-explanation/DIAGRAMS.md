## Class Diagram (UML-lite)

### Simple Class

```
┌─────────────────────────────────────┐
│            ClassName               │
├─────────────────────────────────────┤
│  - field1: Type                     │
│  - field2: Type                     │
├─────────────────────────────────────┤
│  + method1(): ReturnType            │
│  + method2(param: Type): ReturnType │
│  - privateMethod(): void            │
└─────────────────────────────────────┘
```

### Class with Inheritance

```
         ┌───────────────────┐
         │    ParentClass    │
         ├───────────────────┤
         │  + parentMethod() │
         └───────────────────┘
                   ▲
                   │ extends
         ┌─────────┴─────────┐
         │    ChildClass     │
         ├───────────────────┤
         │  - childField     │
         ├───────────────────┤
         │  + childMethod()  │
         └───────────────────┘
```

### Class with Interface

```
    ┌────────────────────┐       <<interface>>
    │    InterfaceName   │◀─────────────
    ├────────────────────┤
    │ + interfaceMethod()│
    └────────────────────┘
              △
              │ implements
    ┌─────────┴─────────┐
    │    ClassName     │
    ├───────────────────┤
    │ - field: Type    │
    ├───────────────────┤
    │ + method()       │
    └───────────────────┘
```

### Relationship Types

```
    ┌─────┐         ┌─────┐      ┌─────┐        ┌─────┐
    │  A  │─────────│  B  │      │  C  │◆───────│  D  │
    └─────┘  uses   └─────┘      └─────┘ has-a └─────┘
                                        │
    ┌─────┐         ┌─────┐              ▼
    │  E  │◇────────│  F  │         ┌─────┐
    └─────┘ has-many└─────┘         │  G  │
                                     └─────┘
    Legend:
    ────  Association (uses)
    ─────▶ Composition (owns, one-to-many)
    ◆────▶ Aggregation (has-a)
```

## Box-Drawing Characters Reference

```
┌ ─ ┐ │ └ ┘ ┌
─    ─    ─    ─
│    │    │    │
├    ├    ┌    ┤
┬    ┴    ┘    ┼
```

### Common Box Characters
- `┌` `┬` `┐` - Top corners and junctions
- `└` `┴` `┘` - Bottom corners and junctions
- `├` `┼` `┤` - Vertical junctions
- `─` - Horizontal lines
- `│` - Vertical lines
- `┴` - T-junction pointing down
- `┬` - T-junction pointing up
- `┤` - T-junction pointing left
- `├` - T-junction pointing right
